What is DOM Object Model

The Document Object Model is a programming interface that represents an HTML document as a tree of objects. Each element and attribute in the document becomes a node that can be operate using JavaScript.

How do you select an Element from DOM

What is event delegation in the context of the DOM,and what is it useful?

Event delegation is a technique in JavaScript used to manage events efficiently by leveraging the event bubbling mechanism in the DOM. Reduces the number of event listeners in your application. Instead of adding listeners to many child elements, you only add one to the parent. This is particularly beneficial when dealing with a large number of elements.
